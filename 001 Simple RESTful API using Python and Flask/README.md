## Run the flask
```
python app.py
* Running on http://127.0.0.1:5000/
* Restarting with reloader
```
## Invoke from curl
To get all task
```
curl http://localhost:5000/todo/api/v1.0/tasks
```
To get single task
```
curl http://localhost:5000/todo/api/v1.0/tasks/2
```
